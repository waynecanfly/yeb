# yeb
办公系统：前后端分离模式，前端使用vue框架+element ui,后端使用gin框架实现。enjoy~

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
